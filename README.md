# Bitcoin Ticker
Sample App demonstrating  subscription to Websocket streams, implemented using MVVM and RxSwift

## Features
1. MVVM Design Pattern
2. Dependency Injection with Swinject
3. RxSwift
4. Unit Tests


## Libraries Used:
1. Swinject + SwinjectStoryboard
2. RxSwift / RxCocoa
3. SwiftWebSocket
4. Lottie
